# JSTQB CBT Simulator

JSTQB（ソフトウェアテスト技術者資格）のCBT（Computer Based Testing）試験を模擬するためのWebアプリケーションです。
React + Vite で構築されており、独自の問題セットを作成・練習できます。

![License](https://img.shields.io/github/license/n4tt0u/jstqb-cbt-simulator)

## 🚀 サイト

**[https://jstqb-cbt-simulator.vercel.app](https://jstqb-cbt-simulator.vercel.app)**

## ✨ 主な機能

- **一問一答モード**: 一問一答形式で、即座に正解・不正解と解説を確認できます
- **本番モード**: 全問題に解答後に結果を表示します
  - ⏳ **タイマー機能**: 制限時間を設定可能（デフォルトは無制限）
  - 🚩 **後で見直すフラグ**: 迷った問題にフラグを立てて、後で一覧から確認・ジャンプできます
  - ❖ **問題一覧**: 本番モードでは問題番号をクリックして任意の問題にジャンプ可能
  - ⌨️ **キーボードショートカット**: 矢印キーで選択肢選択・問題移動、Fキーでフラグ切り替え
- **結果分析**: 正答率、各問題の解説、経過時間を確認可能
- **復習用エクスポート**: 不正解問題やフラグ付き問題をCSV形式でエクスポート可能
- **ダークモード**: 目に優しいダークテーマに対応（設定はブラウザに保存）
- **問題データのインポート**:
  - **デフォルト問題**: あらかじめ用意された問題セットをロード可能
  - **CSVファイル**: 画面上から任意のCSVファイルをアップロード
  - **AnkiNLM対応**: AnkiNLMからコピーしたJSON形式の問題をクリップボード経由でインポート

## 🚀 使い方

### 1. 起動方法

```bash
# 依存関係のインストール
npm install

# ローカルサーバーの起動
npm run dev
```

ブラウザで `http://localhost:5173` にアクセスします。

### 2. 問題データの形式

CSVファイルは以下の形式にします（ヘッダー行が必要）:

```csv
question_text,option_a,option_b,option_c,option_d,correct_option,explanation
"問題文", "選択肢A", "選択肢B", "選択肢C", "選択肢D", "a", "解説文"
```

- `question_text`: 問題文
- `option_a` ～ `option_d`: 選択肢（4択）
- `correct_option`: 正解の記号 (`a`, `b`, `c`, `d`) ※小文字
- `explanation`: 解説文

### 3. 演習の流れ

1. **Step 1: 問題データの準備**
   - デフォルト問題集をロード、またはCSVファイルをアップロード、またはAnkiNLMからインポート
2. **Step 2: 演習モードを選択**
   - 一問一答モード: 1問ごとに正解・不正解と解説を表示
   - 本番モード: 全問題解答後に結果を表示（制限時間設定可能）

### 4. 操作方法

**キーボードショートカット:**
- `↑` `↓`: 選択肢の選択
- `←` `→`: 前の問題・次の問題へ移動
- `F`: 「後で見直す」フラグのオン/オフ

**マウス操作:**
- 選択肢をクリックして回答
- 「後で見直す」をクリックしてフラグを設定
- 本番モードでは「❖ 問題の選択」ボタンで問題一覧を表示

## 🛠️ 技術スタック

- **Frontend**: React 19.2
- **Build Tool**: Vite 7.2
- **Styling**: Pure CSS（CSSフレームワーク不使用）
- **Data Parsing**: PapaParse 5.5
- **Linting**: ESLint 9

## 📦 デプロイ (公開)

ビルドコマンドを実行すると `dist` フォルダが生成されます。
このフォルダを Vercel や Netlify などのホスティングサービスにアップロードするだけで公開可能です。

```bash
npm run build
```
