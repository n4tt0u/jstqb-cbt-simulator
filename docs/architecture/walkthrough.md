# プロジェクト把握 / ウォークスルー

## 調査結果まとめ

このプロジェクトは、JSTQB試験対策のためのCBTシミュレーターです。
現在、基本的​​な試験機能は実装されており、問題データは外部CSVファイルで管理されています。

### 主な機能フロー

1. **起動時**: `StartScreen` が表示され、サーバー(public)から `questions.csv` を取得します。
2. **試験開始**: ユーザーがモードを選択すると `App.jsx` の状態が更新され、`QuestionScreen` に遷移します。
3. **解答中**:
    - `Previous` / `Next` ボタンで問題移動。
    - 選択肢をクリックして回答。
    - 画面右上のフラグで「後で見直す」マーク。
    - **タイマー**: 試験モード時、残り時間が表示されます。
      - 通常: 黒色
      - **残り1分未満: 黄色（警告）**
      - 0秒超過: 赤色
4. **終了**: 「試験終了」ボタンまたは時間切れで `ResultScreen` に遷移。
5. **結果確認**: 正答数、正答率、合否（65%ラインなど）が表示され、各問題の解説を確認できます。

### ファイル構成のポイント

- **`src/App.jsx`**: アプリケーションの"脳"。画面の切り替えロジックと、回答データの保持を行っています。
- **`public/questions.csv`**: 問題データベース。ここを編集するだけで問題の追加・修正が可能です。
- **`src/hooks/useExamTimer.js`**: タイマーロジックが分離されており、再利用性が高められています。
